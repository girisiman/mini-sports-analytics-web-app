# mini-sports-analytics-web-app
A Flask-based sports analytics dashboard demonstrating the complete data science workflow: data cleaning, exploratory data analysis, visualization, and web-based reporting.
_ _ _
# 🏛️ Academic Context
This repository was developed for students at Herald College, Kathmandu pursuing courses in Data Analytics and Visualization (2026). It serves as a practical learning resource delivered by the Herald College Centre for AI and Emerging Technologies, bridging theoretical concepts with real-world implementation.
# 🚀 Features
    📈 Interactive Dashboards - Real-time data visualization with filters
    
    🔍 Exploratory Analysis - Automated statistical summaries
    
    📊 Multiple Visualization Types - Bar charts, scatter plots, heatmaps, and more
    
    🎛️ Dynamic Filtering - Interactive data exploration controls
    
    📱 Responsive Design - Accessible on desktop and mobile devices
    
    💾 Data Export - Download analysis results in various formats
