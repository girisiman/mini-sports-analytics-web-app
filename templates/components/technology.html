Technology Stack

How the Application Works

The Mini Sports Analytics Dashboard combines multiple technologies, each responsible for a different part of the analytics workflow.

Backend

Handles routing, application logic, and communication between the browser and Python.

Python Flask

Data Analytics

Reads CSV files, cleans data, computes statistics, and prepares information for visualization.

Pandas NumPy

Visualization

Produces publication-quality static charts for communicating analytical insights.

Matplotlib Seaborn

Frontend

Presents processed data, tables, charts, and dashboard components to end users.

Bootstrap 5 HTML5 CSS3 JavaScript
